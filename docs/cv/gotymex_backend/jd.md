# About the Team

Our team is responsible for creating a brand-new suite of digital financial platforms and applications that are highly reliable and scalable using modern engineering practices.

You will join the **Digital Bank Build Team**, responsible for designing, building, and maintaining the digital banking platform and its customer channels.

As a Backend Engineer, you will:

- Implement software features.
- Participate in technical design discussions.
- Write automated tests to ensure high-quality product delivery.
- Work in Scrum and DevOps environments.

The interview process focuses on evaluating your coding and system design skills while providing an engaging technical experience.
